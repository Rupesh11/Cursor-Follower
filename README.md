Custom Cursor Effect

An interactive custom cursor built using JavaScript, HTML, and CSS.

Features

- Cursor follows mouse movement in real-time
- Expands on movement and shrinks when idle
- Displays dynamic text inside the cursor
- Smooth animations using CSS transitions

Technologies Used

- JavaScript (DOM manipulation & events)
- HTML
- CSS (animations & styling)

How It Works

The cursor position is updated using the `mousemove` event.  
When the mouse moves, the cursor grows and displays text.  
After a short delay (300ms), it shrinks back and hides the text.
